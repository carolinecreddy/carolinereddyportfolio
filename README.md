# Caroline Reddy Portfolio Website

A clean, responsive professional portfolio website for Caroline Reddy, built for a GitHub Pages portfolio assignment.

## Files
- `index.html` - website structure and content
- `styles.css` - full visual design and responsive layout
- `script.js` - mobile navigation behavior and footer year
- `assets/caroline-reddy-headshot.png` - professional headshot
- `assets/Caroline-Reddy-Resume.pdf` - downloadable resume

## How to publish with GitHub Pages
1. Create a GitHub account at github.com.
2. Create a new public repository named `caroline-reddy-portfolio` or `caroline-personal-website`.
3. Upload all files and folders from this project into the repository.
4. Go to Settings > Pages.
5. Under "Build and deployment," choose "Deploy from a branch."
6. Select the `main` branch and `/root`, then save.
7. Wait a minute or two. Your site should publish at a URL like:
   `https://YOUR-GITHUB-USERNAME.github.io/caroline-reddy-portfolio/`
